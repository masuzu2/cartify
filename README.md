## ที่มา

งานชิ้นนี้ไม่มี backend จริงครับ ข้อมูลทั้งหมดเก็บไว้ใน `localStorage` ของเบราว์เซอร์ — ลบประวัติเมื่อไรก็หายเมื่อนั้น เหมาะเอาไปเป็นเดโม่ ส่งอาจารย์ หรือเป็น template ต่อยอด

## ✦ รันยังไง

โหลดมาเปิด `index.html` ได้เลย ไม่ต้องลง dependencies

```bash
git clone https://github.com/masuzu2/cartify.git
cd cartify
open index.html     # macOS
# start index.html  # Windows
```

ถ้าอยากให้ PWA/Service Worker ทำงานครบ แนะนำให้รันผ่าน local server

```bash
python3 -m http.server 8000
# หรือ
npx serve .
```

แล้วเปิด <http://localhost:8000>

เข้าหน้า admin โดยต่อท้าย URL ด้วย `?admin=1`
